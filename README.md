## 🧰 Technologies Used

### 🖥️ Frontend

| Technology | Description |
|-----------|-------------|
| [**React (Vite)**](https://vitejs.dev/) | A modern JavaScript framework used to build fast and responsive user interfaces. Vite provides blazing fast hot module replacement (HMR) and an optimized build process. |
| [**Tailwind CSS**](https://tailwindcss.com/) | A utility-first CSS framework that allows for rapid UI development with custom designs directly in the markup. |
| [**Boxicons**](https://boxicons.com/) | A simple and lightweight icon library to add scalable icons in your UI. |
| [**React Router DOM**](https://reactrouter.com/) | Handles client-side routing, allowing for a single-page application (SPA) experience with multiple views. |

---

### ⚙️ Backend

| Technology        | Description                                                                                   |
|-------------------|-----------------------------------------------------------------------------------------------|
| [**Node.js**](https://nodejs.org/)       | A JavaScript runtime built on Chrome's V8 engine, used for building scalable network applications.         |
| [**Express.js**](https://expressjs.com/) | A minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications. |
| [**MongoDB**](https://www.mongodb.com/)  | A NoSQL database used to store and retrieve data in a flexible, JSON-like format.                           |
| [**JWT (JSON Web Tokens)**](https://jwt.io/) | Used for secure authentication and authorization between frontend and backend systems.              |
| [**Bcrypt.js**](https://www.npmjs.com/package/bcryptjs) | A library to hash user passwords for secure storage.                                         |
| [**Mongoose**](https://mongoosejs.com/) | Elegant MongoDB object modeling for Node.js, making it easier to manage schemas and relationships.         |
| [**Dotenv**](https://www.npmjs.com/package/dotenv) | For managing environment variables securely (e.g., secrets, DB URLs, API keys).           |

## 📌 Setup Instructions

### 🔧 Frontend
```bash
cd client
npm install
npm run dev
````

### ⚙️ Backend

```bash
cd server
npm install
npm start
```

> Make sure to create a `.env` file in the `server` directory with your MongoDB URI and JWT secret.
